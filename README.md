## 📌 Overview

AstraAir is an AI-powered environmental intelligence platform that combines weather information, air quality monitoring, satellite observations, environmental analytics, and machine learning into one interactive dashboard.

The platform helps users understand environmental conditions using modern visualizations and AI-driven insights.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Dinesh-cloud2/AstraAir.git
```

Move into the project

```bash
cd AstraAir
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
